# 项目简介

## 核心功能
- 首页推荐
- 课程信息管理
- 资料信息管理
- 公告信息发布与查看
- 论坛交流
- 新闻资讯
- 个人中心
- 管理员主页面
- 教师课程管理
- 学生学习进度与评论查看

## 适用场景
- 考研学习交流
- 教师教学管理
- 学生学习资源获取

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
